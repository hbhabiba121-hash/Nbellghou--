# Nballgou

Nballgou is a civic platform that allows citizens to report urban issues in Rabat.

## Features

- Report city problems
- Upload photos
- Interactive Rabat map
- Track ticket status
- Admin dashboard

## Tech Stack

- React / Next.js
- Tailwind
- Mapbox
- Node.js

## Vision

Inspired by Rabat Smart City and urban digitalization initiatives.
